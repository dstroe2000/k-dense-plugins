# Skill Disambiguation — Protein Engineering

## 3 skills — NO OVERLAP

| Skill | Use when |
|-------|----------|
| esm | Protein language models (ESM3, ESM C) — sequence/structure/function prediction |
| glycoengineering | Glycosylation analysis — N-glycosylation sequons, O-glycosylation prediction |
| adaptyv | Adaptyv Bio Foundry API — submit protein experiments, retrieve results |

Each targets a different stage: `esm` designs proteins, `glycoengineering` analyzes modifications, `adaptyv` runs physical experiments.
