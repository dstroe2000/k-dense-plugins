# Skill Disambiguation — Thinking & Decision Tools

## Research Ideation (5 skills — MEDIUM OVERLAP)

These all help "think about things" but from different angles:

```
"Help me think about / explore X"
  │
  ├─ Open-ended brainstorming, no data yet?
  │   └─ scientific-brainstorming (in k-dense-writing)
  │
  ├─ Have observations, need hypotheses?
  │   └─ hypothesis-generation (in k-dense-writing)
  │
  ├─ Want multiple persona perspectives on a decision?
  │   └─ consciousness-council — Mind Council, diverse viewpoints
  │
  ├─ "What if X happened?" scenario analysis?
  │   └─ what-if-oracle — structured branching, possibility space
  │
  ├─ Have tabular data, want automated pattern discovery?
  │   └─ hypogenic — LLM-driven hypothesis testing on datasets
  │
  ├─ Analyze someone's thinking/writing style?
  │   └─ dhdna-profiler — cognitive pattern extraction
  │
  └─ Market/industry analysis?
      └─ market-research-reports — McKinsey/BCG style, 50+ pages
```

### Key Distinctions

| Skill | Input | Output | Method |
|-------|-------|--------|--------|
| consciousness-council | Any question | Multi-perspective deliberation | Panel of personas |
| what-if-oracle | "What if..." scenario | Branching possibility tree | Structured analysis |
| hypogenic | Tabular dataset | Ranked hypotheses | LLM + data testing |
| dhdna-profiler | Text sample | Cognitive fingerprint | Pattern extraction |
| market-research-reports | Industry/market topic | Consulting-style report | Research + formatting |

**Note**: `scientific-brainstorming` and `hypothesis-generation` live in `k-dense-writing` but overlap conceptually. The split:
- brainstorming = divergent, no data needed, exploring
- hypothesis-generation = convergent, observations in hand, formalizing
