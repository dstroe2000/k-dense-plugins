# Skill Disambiguation — Writing & Communication

## Research Methodology (14 skills)

### Writing vs Reviewing vs Planning

```
"I need to write/review/plan research"
  │
  ├─ Writing a manuscript?
  │   └─ scientific-writing — IMRAD structure, flowing prose, citations
  │
  ├─ Reviewing someone else's manuscript?
  │   └─ peer-review — checklist evaluation, CONSORT/STROBE, constructive feedback
  │
  ├─ Evaluating scholarly work quality?
  │   └─ scholar-evaluation — ScholarEval framework, quantitative scoring
  │
  ├─ Writing a grant proposal?
  │   └─ research-grants — NSF/NIH/DOE/DARPA templates, budget, broader impacts
  │
  └─ Managing citations?
      ├─ Need BibTeX / DOI validation → citation-management
      └─ Working with Zotero library → pyzotero
```

### Ideation & Thinking

```
"I need ideas / brainstorming"
  │
  ├─ Open-ended creative exploration?
  │   └─ scientific-brainstorming — interdisciplinary connections, research gaps
  │
  ├─ Have observations, need testable hypotheses?
  │   └─ hypothesis-generation — structured, predictions, experiment design
  │
  └─ Need to evaluate claims or evidence?
      └─ scientific-critical-thinking — bias detection, GRADE, evidence grading
```

### Presentations & Posters

```
"I need to present research"
  │
  ├─ Slide deck / conference talk?
  │   ├─ Research-focused → scientific-slides
  │   └─ General PowerPoint work → pptx (in k-dense-search plugin)
  │
  ├─ Research poster?
  │   ├─ LaTeX (default, best quality) → latex-posters
  │   └─ Explicitly need PPTX format → pptx-posters
  │
  └─ Need journal/conference formatting?
      └─ venue-templates — Nature, Science, IEEE, NeurIPS templates
```
